# Changelog

All notable changes to the 2026 World Cup Knockout Stage Tracker.

---

## [1.4.0] — 2026-06-25 · Light theme + layout resize

### Changed
- Switched from dark to light theme throughout (white cards, slate text, soft colored badges)
- Reduced layout to 75% width centered — better readability on wide screens
- W/D/L buttons updated to use soft colored fills on active state
- Status badges now use light-mode bordered pill style
- Scotland highlight changed from dark purple to soft blue

---

## [1.3.0] — 2026-06-25 · Live results update + confirmed eliminations

### Changed
- Updated Group A, B, C results after matchday 3 completion
- Mexico confirmed Group A winner (9 pts, GD +5) — added to 3rd place table
- South Africa upset South Korea 1–0 — South Africa now 3rd in Group A
- Czechia confirmed eliminated (lost 0–3 to Mexico)
- Qatar confirmed eliminated
- Added amber warning banner for still-live matches (Ecuador vs Germany, Curaçao vs Ivory Coast)
- Updated notes on all Group A, B, C teams

---

## [1.2.0] — 2026-06-25 · All 48 teams + tabbed layout

### Added
- Tab 1: All 48 teams view across all 12 groups
- Five-tier status system: Confirmed Through, Likely Through, On the Bubble, Likely Out, Confirmed Eliminated
- Filter pills to isolate teams by status category
- Notes column per team with context on their situation
- Tab 2 now contains the 3rd place simulator (previously the only view)

### Changed
- Layout restructured from two-panel side-by-side to full-width tabbed interface

---

## [1.1.0] — 2026-06-25 · Win probabilities added

### Added
- Betting odds (bet365 matchday 3) integrated into 3rd place simulator
- W / D / L percentage shown per match
- Percentage labels on each scenario button
- Defaults now set to most likely outcome per odds (not always Win)
- "Show odds" toggle to declutter the table
- Scotland rank updates live in header pill as scenarios change

### Changed
- Scenario buttons now show probability % below letter label
- Table layout expanded to include odds column when visible

---

## [1.0.0] — 2026-06-25 · Initial build

### Added
- Widescreen two-panel layout: confirmed qualifiers left, 3rd place simulator right
- 13 confirmed knockout teams listed with winner/runner-up badges
- 5 confirmed eliminated teams
- 3rd place race table with 12 teams, W/D/L scenario buttons
- Live re-ranking by points then goal difference
- Red elimination line separating top 8 from bottom 4
- Scotland highlighted throughout with live status pill
